# 00. AI Ops MVP Charter

## 1) 목적
사내 운영 업무(재고/발주/물류/매출)에서 반복작업을 줄이고,
의사결정을 빠르게 하는 **실사용 가능한 운영 대시보드 + AI 업무 보조 시스템**을 구축한다.

## 2) 타겟 사용자
- 운영 담당자
- 물류/구매 담당자
- 팀 리더(일일 KPI 확인)

## 3) 핵심 가치
1. 가독성 높은 운영 데이터 가시화
2. 재고/발주 상태를 즉시 파악 가능한 관리성
3. 반복 업무 자동화 기반 마련
4. AI 질의로 요약/이상징후/다음 액션 제안

## 4) 성공 기준 (MVP)
- 3분 내 오늘 운영상태 파악 가능 (KPI + 위험상품)
- 발주 생성/조회/입고반영이 한 화면 흐름에서 가능
- 품절 임박/품절 항목이 자동 분류되어 노출
- 일간 리포트 생성에 필요한 데이터 API 확보

## 5) 비목표 (MVP 제외)
- 복잡한 WMS/ERP 전체 대체
- 실시간 이벤트 스트리밍 분석
- 고급 권한체계(RBAC 완성형)

## 6) 기술 원칙
- Frontend: Next.js
- Backend: FastAPI
- DB: PostgreSQL
- Queue: Redis + worker (선택)
